从表 \ref{tab:datasets}可以看出，代表性语音情绪数据集首先根据情绪采集方式与情绪表达性质被较为明确地区分。 RAVDESS、EMO-DB、SAVEE、TESS、CREMA-D、CASIA等数据集属于在受控环境下由演员表达情绪的表演型基准数据集，这些数据集的共同特点是情绪标签相对明确、实验条件较为稳定。 这类数据集较为便利于模型间性能比较与算法基线构建，并有助于保障基础实验的可重复性，因此在语音情绪识别研究中长期作为代表性基准数据集使用。 相比之下，IEMOCAP与MSP-IMPROV包含对话与即兴互动，比表演型基准数据集具有相对更为复杂的情绪表达； 而MSP-Podcast、FAU Aibo、EMOVOME、CNSCED等数据集则包含自然语音或实际环境因素，具有更接近实际应用场景的特性。 然而，这类自然型数据集也同时面临情绪边界模糊、标签一致性以及数据整理难度较高等问题。

语音情绪数据集还可以根据标签构成方式与问题定义方式进行区分。RAVDESS、EMO-DB、SAVEE、TESS、CREMA-D、CASIA等数据集提供愤怒、快乐、悲伤、恐惧、中性等离散标签，因此较为便利地应用于基于多分类的实验。相比之下，RECOLA、MSP-Podcast、EMOVOME等数据集则同时提供效价、唤醒度、支配度等连续维度，因此较为适用于将情绪作为回归问题或维度情感模型处理的研究。此外，对于IEMOCAP、MELD这类对话上下文重要的数据集，除了发话的声学特征外，还可以进一步考虑说话人交替、上下文流程、对话结构等附加因素。上述分析表明，数据集之间的差异不仅体现在规模上，也与研究如何定义情绪以及设定何种任务存在一定关联[3,11]。

另一方面，语言、说话人构成、公开访问方式也是用于比较数据集时的主要标准。 英语数据集不仅包括RAVDESS、TESS、CREMA-D、SAVEE等表演型基准数据集，还较为常用IEMOCAP、MSP-IMPROV、MSP-Podcast、MELD等对话型、自然型数据集。 德语方面有EMO-DB与FAU Aibo，汉语方面有CASIA与CNSCED，波斯语方面有ShEMO，西班牙语方面有EMOVOME，均为代表性示例。 说话人数与人口统计信息在不同数据集之间也存在一定差异。例如，TESS仅包含两名女性说话人，SAVEE仅包含男性说话人，而CREMA-D则包含91名具有不同年龄和人口统计信息的说话人。 MSP-Podcast在说话人数与发话多样性方面属于较为大规模的自然语音数据集，因而在一定程度上区别于其他基准数据集。 此外，部分数据集相对易于公开访问，部分则需要许可证注册或单独审批程序，部分数据集仅选择性使用原始多模态资料中的音频子集。 数据集在语言、人口统计信息与公开访问方式上的差异，不仅体现在附加信息层面，也与研究的可重复性与泛化能力密切相关。

代表性语音情绪数据集在情绪采集方式、标签构成方式、语言、说话人构成、公开访问方式等方面存在一定差异。这些差异不仅体现在数据规模上，也与后续的特征提取、模型设计、实验条件设定及结果解读密切相关。因此，数据集的选择不仅取决于引用频次或使用便利性，更应考虑其是否符合研究目的与实验条件。为此，本研究在数据集选择时重点考虑了以下几项标准。其一，数据集宜具备公开访问性或在研究过程中可稳定获取，且宜具备支持后续实验可重复性的形式。其二，标签构成方式宜相对明确，能够直接应用于分类实验。其三，说话人数与性别分布不宜过度偏斜，宜具备基本的平衡性与代表性。其四，数据规模宜能够支撑基础实验，同时预处理与模型验证不宜过于复杂。其五，数据集宜具备基准数据集性质，即在已有研究中较为常用，以便后续进行结果比较与实验分析。基于上述标准，本研究在初期实验阶段，相比于结构复杂的对话型数据集或连续维度的数据集，优先考虑适用于分类实验且结构较为清晰的数据集。数据集选择标准的制定，不仅是对研究范围的明确，也为构建可验证的研究设计提供了基础。

基于上述标准，本研究首先选择RAVDESS作为基础实验数据集。RAVDESS具有公开访问，性别分布较为均衡，情绪类别构成清晰，因此较为适合进行基础分类实验。此外，RAVDESS在语音情绪识别相关现有研究中较为常用，属于代表性基准数据集之一，较为便利于后续实验结果的分析与比较。RAVDESS属于在受控环境下由专业演员演绎情绪的表演型基准数据集，与实际应用场景中的自然语音存在差异。然而，综合考量数据访问性、标签稳定性、实验可重复性、输入处理流程的清晰度等因素，该数据集被认为适合在本研究的初期实验阶段使用。


\section{信号处理技术}

\subsection{信号预处理}

\textbf{输入格式统一：重采样与通道统一}

不同语音数据集或录音设备采集的音频文件可能具有不同的采样频率和通道配置。 部分文件以8 kHz、16 kHz、22.05 kHz、44.1 kHz或48 kHz等不同采样频率存储，且单声道与立体声格式也可能混用。 这些差异对可分析的频率范围、时间分辨率以及后续特征的统计分布可能产生一定影响，因此为稳定应用相同的特征提取流程，需要首先统一输入格式。 为此，通常需要先进行重采样与通道统一。

重采样（resampling）是将原始采样频率下的离散信号重新表达为目标采样频率的过程。概念上可理解为先从原离散信号恢复连续时间信号，再按目标采样频率重新采样，理想化情况下可表示为：

\[
x_r[m] = x_c\left(\frac{m}{F_s'}\right)
\]
其中 $x_r[m]$ 为重采样后的离散信号，$x_c(t)$ 为恢复的连续时间信号，$F_s'$ 为目标采样频率，$m$ 为新采样索引。

实际实现中并不直接计算连续时间信号，而是通过插值与抗混叠滤波相结合的数字化处理完成。尤其在下采样时，超出奈奎斯特频率的成分可能引发混叠，通常需通过低通滤波器抑制高频成分后再进行重采样。对于立体声或多声道输入信号，为保持特征提取的一致性，常将其转换为单声道。一种常见的方法是取左右声道平均值，可表示为：

\[
x_{\text{mono}}[n] = \frac{x_L[n] + x_R[n]}{2}
\]
其中 $x_L[n]$ 和 $x_R[n]$ 分别为左右声道信号，$x_{\text{mono}}[n]$ 为转换后的单声道信号。该方法实现简单，较为适用于多数语音情绪识别实验，但当某些声学信息仅集中于特定声道时，可能因平均化导致信息损失。

重采样与通道统一虽不直接产生情绪线索，但对于保证后续特征提取输入条件的一致性较为重要。若采样频率与通道格式未统一，即便采用相同的 STFT 或 Mel 滤波器组参数，也可能得到分布不同的特征，从而增加模型误将设备差异或存储格式差异作为情绪信息学习的风险。因此，统一输入格式有助于保障后续处理可重复性与实验可比性的基础环节。

\textbf{振幅归一化}

不同语音文件在说话人的发音音量、麦克风灵敏度、录音距离、设备增益、存储格式等方面可能存在一定差异。 这种差异可能源于情绪表达本身的差别，但在许多情况下也可能反映录音条件的不同。若保留这种振幅差异直接进行特征提取，模型可能过度学习与情绪无关的振幅变化，从而影响识别性能[2,3,6]。 因此，需要对输入信号进行振幅归一化（amplitude normalization），使其振幅范围保持一致。

峰值归一化（peak normalization）是一种简单且常用的方法之一。 该方法以整个信号中绝对值最大的样本为基准进行缩放，可表示为：

\[
x_{\text{peak}}[n] = \frac{x[n]}{\max_{0 \le n < N} |x[n]| + \varepsilon}
\]
其中 $x[n]$ 为原始信号，$x_{\text{peak}}[n]$ 为峰值归一化后的信号，$N$ 为总样本数，$\varepsilon$ 为防止分母为零的极小常数。经此归一化的信号，其最大绝对振幅通常较为接近1。峰值归一化的优点在于计算简单、直观，能快速将信号动态范围调整到统一水平。峰值归一化的缺点在于，若信号中存在较大的瞬时峰值，其余部分的相对强度可能被过度压缩[6]。

均方根归一化（RMS normalization）也是一种常用的方法之一。均方根（RMS）反映信号的平均强度水平，定义为：

\[
\mathrm{RMS}(x) = \sqrt{\frac{1}{N}\sum_{n=0}^{N-1} x[n]^2}
\]
据此，RMS 归一化可表示为：
\[
x_{\text{rms}}[n] = \frac{x[n]}{\mathrm{RMS}(x) + \varepsilon}
\]
其中 $\mathrm{RMS}(x)$ 为原始信号的均方根值，$x_{\text{rms}}[n]$ 为归一化后的信号。若需要指定目标 RMS 值 $r_t$，可表示为：
\[
x_{\text{target-rms}}[n] = \frac{r_t}{\mathrm{RMS}(x) + \varepsilon} x[n]
\]
其中$r_t$为目标 RMS 值。RMS 归一化以信号的均方根值为基准，相较于峰值归一化，通常能提供较为稳定的幅度调节效果[4,5,6]。

在语音情绪识别中，振幅归一化有助于减少因录音条件差异带来的无关振幅变化。 然而，情绪表达本身也包含发声强度、响度等重要线索，过度归一化可能削弱这些与情绪相关的强度信息。 例如，愤怒、喜悦常伴随较高发声强度，而悲伤、疲惫则可能呈现较低且平缓的强度分布。 因此，应用振幅归一化时，应在消除录音条件差异与保留情绪相关的强度信息之间取得平衡[3, 4,6]。

此外，振幅归一化并非适用于所有情况。将整个信号缩放至统一振幅范围时，背景噪声也会被同步放大或缩小。因此，振幅归一化可视为信号振幅统一的基础步骤，噪声消除与鲁棒性增强宜结合后续的数据增强或鲁棒性策略共同实现。
